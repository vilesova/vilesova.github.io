Главное меню
============

Служит входной точкой при запуске игры.


Кнопки
--------------------------------

Для главного меню существует несколько кнопок, функционал которых описан ниже:

Играть
~~~~~~~~~~~
 #. Нажмите кнопку «играть», чтобы запустить игру

Рекорды
~~~~~~~~~~~
 #. Нажмите кнопку «рекорды», чтобы открыть рекорды

Настройки
~~~~~~~~~
 #. Нажмите кнопку «настройки», чтобы открыть настройки
